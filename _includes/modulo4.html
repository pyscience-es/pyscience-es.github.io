Módulo 4: Módulos Básicos

Mäia ha sido capaz de salir ilesa del campo de flores y se para a preguntar la dirección hacia la pirámide a una araña gigante, pero la araña tiene otros planes para nuestra aventurera. Para conseguir escapar de las redes de este monstruo peludo tendrás que aplicar tus conocimientos de los módulos básicos de Python. Se hará una introducción a numpy, una librería de cálculo científico con gran potencia de cálculo matricial; pandas, librería para el análisis estadístico de datos; y seaborn, una librería que permite la visualización gráfica de resultados de forma sencilla.
