Módulo 5: Aplicación en Python

Por fin, Mäia ha conseguido llegar a la base de la pirámide. Ahora deberá escalar hasta la cima para conseguir encontrar el tesoro oculto. En esta sección se realizará una introducción a PyQt, una librería basada en Qt que permite realizar interfaces gráficas. También se realizará una introducción a 3D Slicer, plataforma de imagen médica que integra distintos tipos de módulos para el tratamiento de datos y se enseñará a desarrollar módulos para la misma.
